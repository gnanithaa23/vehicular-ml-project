# 🚗 Vehicular Prediction System (ECLS 4.0)

## 📌 Project Overview

This project is a web-based vehicular prediction system that uses an **ensemble of machine learning and deep learning models** to analyze vehicle data and predict conditions accurately. By combining multiple models, the system improves prediction performance compared to single-model approaches.

---

## 🚀 Features

* 🔹 Predicts vehicle conditions using input data
* 🔹 Ensemble model (Deep Learning + Random Forest)
* 🔹 User-friendly web interface built with Django
* 🔹 Real-time prediction results
* 🔹 Improved accuracy using hybrid modeling

---

## 🛠 Tech Stack

* **Backend:** Python, Django
* **Machine Learning:** TensorFlow / Keras, Scikit-learn
* **Frontend:** HTML, CSS
* **Dataset:** CSV files

---

## 🧠 How It Works

1. User enters vehicle-related data through the web interface
2. Input data is preprocessed
3. Data is passed to trained ML/DL models
4. Ensemble model combines predictions
5. Final result is displayed to the user

## ▶️ How to Run the Project

### Step 1: Clone the repository

```
git clone https://github.com/gnanithaa23/vehicular-ml-project.git
cd vehicular-ml-project
```

### Step 2: Install dependencies

```
pip install -r requirements.txt
```

### Step 3: Run the server

```
python manage.py runserver
```

### Step 4: Open in browser

```
http://127.0.0.1:8000/
```
